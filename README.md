# Last-Web-Server
Web server made in java
<br />
<u> How to launch </u>
- Navigate to the web server folder
- Go to the bin folder
- Open a terminal in the bin folder
- Execute Start.sh
<br />
<h3>Functionality </h3>

- Can handle post and get request
- Can read Image
- Can read php file
<br />

<ul>
<li>

</li>

</ul>
